# Code of Conduct

This project adopts the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) as its code of conduct. All contributors, maintainers, and participants are expected to uphold these standards.

For the full text of the code of conduct, see: https://www.contributor-covenant.org/version/2/1/code_of_conduct/

## Reporting Issues

If you experience or witness unacceptable behavior, contact the maintainers:

- https://github.com/pedro-angel/kibana-py/issues

All reports are handled confidentially and reviewed promptly.
