# Maintainers

This file defines ownership for `kibana-py` release and maintenance workflows.

## Current Maintainers

- [@pedro-angel](https://github.com/pedro-angel)

## Responsibilities

- Review and merge pull requests
- Maintain release notes and versioning
- Approve and monitor PyPI releases
- Maintain CI/CD, security checks, and dependency policy
- Triage issues and security reports

## Release Ownership

- Releases are cut from protected `main` only.
- Release tags must use `vX.Y.Z`.
- Maintainers verify changelog, tests, and package integrity before tagging.
